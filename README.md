# 🦒 Giraffe Zoo - Animal Welfare & Conservation Initiative

A comprehensive web platform dedicated to promoting animal welfare in zoos, raising awareness about conservation efforts, and providing resources for reporting and tracking animal welfare incidents.

## Project Overview

Giraffe Zoo is a modern, user-friendly website designed with a gummy aesthetic to make important information about animal welfare accessible and engaging. The platform includes:

- **Educational Content**: Information about zoos' role in animal welfare and conservation
- **Zoo Finder**: Directory of accredited zoos with animal welfare programs
- **Services**: Consultancy, education programs, and conservation support
- **Incident Reporting**: Confidential system for reporting animal welfare concerns
- **Transparency**: Public tracking of reported incidents and resolutions

## Pages & Features

### 1. **Home Page** (`index.html`)
- Hero section featuring a giraffe eating carrots
- Mission statement: "Happy Animals, Better Zoos"
- Core values: Care, Respect, Protect, Educate
- Impact statistics about modern zoos
- How zoos promote animal welfare
- Call-to-action sections encouraging visitor engagement

### 2. **Find a Zoo** (`find-zoo.html`)
- Searchable directory of accredited zoos
- Filter options by country, accreditation, and specialty
- Zoo cards with detailed information
- Links to zoo websites and additional resources
- Form to suggest new zoos for inclusion

### 3. **Services** (`services.html`)
- Zoo Welfare Consultancy
- Educational Programs
- Research & Data Analysis
- Conservation Support
- Incident Reporting & Response
- Strategic Partnerships
- Partnership tiers (Bronze, Silver, Gold)
- Contact form for inquiries

### 4. **Reported Incidents** (`incidents.html`)
- Statistics on incident tracking
- Recent incident reports with status updates
- Filter tabs (All, Investigating, Resolved, Follow-up)
- Detailed incident cards with descriptions and progress
- Anonymous incident reporting form
- FAQ about the reporting process
- Confidentiality and whistleblower protection information

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No backend server required for static content

### Installation
1. Clone the repository
2. Open `index.html` in your web browser
3. Navigate through pages using the navigation menu

### Local Development
```bash
# Simply open the HTML files in your browser
# For best experience, use a local server:
python -m http.server 8000
# Then visit http://localhost:8000
```

## Incident Reporting System

The incident reporting system allows users to:
- Report anonymously or with contact information
- Categorize concerns (enclosure, health, feeding, etc.)
- Provide detailed descriptions
- Attach evidence (photos, videos, documents)
- Opt-in for updates and follow-up

Each report receives:
- Unique incident ID for tracking
- Investigation timeline
- Public status updates
- Confidentiality protection
